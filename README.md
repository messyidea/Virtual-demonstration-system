基于高通虚拟现实的一个安卓应用
===


Introduction
---
    该应用叫虚拟演示系统。
    举个例子，当老师在幻灯片上放一张图片，学生可以通过该应用现实具体的3D模型。
    这样的话也有利于学生的学习。
    特别是对于复杂的分子模型特别有用。

    模型这种不是直接生成的，那太高端了。。做不到。
    现在模型是内置在应用里面的，如果有服务器的话可以做到云端。
    但是以模型的大小和现在的网速，基本上好像不是很有用。
